Agent Downloads

Your Environment Uses Okta OAuth2 Authentication
Your AppDynamics SaaS environment is integrated with Okta. You must obtain a JWT token with the 'download' scope from your Okta identity provider to download agents.

How to Download Agents

  1. Log into your AppDynamics Controller UI
  2. Navigate to the Download Agent wizard for the agent you need
  3. Select the cURL download method
  4. Copy the generated command (it will include a valid JWT token)
  5. Run the command in your terminal
Security Note: The generated token is valid for a limited time (typically 1 hour). Do not share it or hardcode it into scripts.

Example Command Template

curl -L -O -H "Authorization: Bearer YOUR_JWT_TOKEN_HERE" "https://download.appdynamics.com/download/prox/download-file/AGENT_TYPE/VERSION/FILENAME"

Common Agent Paths

  • Java Agent (RPM): java-jdk8-rpm/25.7.0.37201/AppServerAgent-1.8-25.7.0.37201.rpm
  • Java Agent (ZIP): java-agent/latest
  • Machine Agent (Linux): machine-agent-bundle-64bit-linux/latest
  • .NET Agent: dotnet/current/appdynamics-dotnet-core-ga.zip?license=YOUR_LICENSE_KEY
